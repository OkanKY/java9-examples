# Java 9 New Features Examples and Tutorials

Demonstration for [Set Up Eclipse, IntelliJ And NetBeans For Java 9](https://howtoprogram.xyz/2017/09/24/set-up-eclipse-intellij-and-netbeans-for-java-9/)


## Prerequisites:
* Maven >= 3.3.1
* JDK 9

## Import into NetBeans

File --> New Project --> Maven --> Project with Existing POM --> Next --> Finish
Select project directory and Click "Open Project"



## Running the example
* Click Run Project button (F6) to run the main class
* Or right click on the test method --> Run Focused Test Method
* `mvn test`
